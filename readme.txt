Для работы требуется .NET Desktop Runtime 5.0.1 x86
Скачать можно тут:
https://download.visualstudio.microsoft.com/download/pr/55bb1094-db40-411d-8a37-21186e9495ef/1a045e29541b7516527728b973f0fdef/windowsdesktop-runtime-5.0.1-win-x86.exe

Для игры запустить Game.exe

Цель игры: 
Найти жёлтый блок (сокровище) и разрушить его.

Легенда:
██ - Препятствие. Можно разрушить, за исключением границ пещеры.
▒▒ - Туман. Скрывает клетку под собой.
OO - Игрок.
ХХ - Бомба.

Управление:
W - вверх
A - влево
S - вниз
D - вправо
Shift + <кнопка движения> - установить бомбу на клетку. Бомба взрывается через два хода
R - перезапустить игру
