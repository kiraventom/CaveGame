Для игры запустить WpfUI.exe

Цель игры: 
Найти сокровище и собрать его.

Легенда:
Красная стена - Препятствие. Можно разрушить, за исключением границ пещеры.
Серый квадрат - Туман. Скрывает клетку под собой.
Воин - Игрок.
Бочка - Бомба (чёрная - неактивирована, красная - активирована).
Ваза - Сокровище.

Информация:
- Изначально у игрока в инвентаре 2 бомбы.
- Для прохождения игры необходимо собирать в пещере бомбы.
- Инвентарь игрока указан в заголовке окна консоли.

Управление:
W - вверх
A - влево
S - вниз
D - вправо
ЛКМ по клетке - установка бомбы на клетку. Бомбу можно установить только на соседнюю с игроком клетку
R - перезапустить игру
